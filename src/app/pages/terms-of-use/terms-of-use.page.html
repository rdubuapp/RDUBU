ข้อตกลงในการใช้งาน ข้อตกลงในการใช้งาน Application • ผู้ที่เข้ามาใช้บริการจะต้องยอมรับและปฏิบัติตามข้อตกลงในการใช้งาน application • ผู้อ่านควรใช้วิจารณญาณในการใช้งาน application และปรึกษาแพทย์หรือเภสัชก่อนการใช้ยา • ข้อมูล เนื้อหา รูปภาพ และองค์ประกอบต่างๆที่มีอยู่ใน application ถูกเขียนขึ้นตามเอกสารอ้างอิง ที่ระบุในช่วงเวลาที่เขียนเท่านั้น ผู้ใช้งานต้องยอมรับว่าข้อมูลที่เกิดขึ้นภายหลังหรือข้อมูลในฐานข้อมูลที่แตกต่างกัน อาจทำให้เกิดข้อมูลหรือคำแนะนำที่ต่างออกไปจากที่มีการนำเสนอใน application • ข้อมูล เนื้อหา รูปภาพ และองค์ประกอบต่างๆที่ปรากฏใน application นี้เป็นแหล่งให้ความรู้ คำแนะนำ การใช้ยาปฏิชีวนะอย่างสมเหตุสมผลให้กับบุคลากรทางการแพทย์และประชาชนทั่วไปเท่านั้น หากมีปัญหาโรคอื่นนอกเนื่องจากนี้โปรดปรึกษาแพทย์ผู้เกี่ยวข้องโดยตรง และหากมีปัญหาเรื่องยาให้ปรึกษาเภสัชกรหรือบุคลากรทางการแพทย์ที่เกี่ยวข้อง • ข้อมูล เนื้อหา รูปภาพ และองค์ประกอบต่างๆที่ปรากฏใน application นี้เป็นแหล่งให้ความรู้ คำแนะนำ การใช้ยาปฏิชีวนะอย่างสมเหตุสมผลให้กับบุคลากรทางการแพทย์และประชาชนทั่วไปเท่านั้น หากมีปัญหาโรคอื่นนอกเนื่องจากนี้โปรดปรึกษาแพทย์ผู้เกี่ยวข้องโดยตรง และหากมีปัญหาเรื่องยาให้ปรึกษาเภสัชกรหรือบุคลากรทางการแพทย์ที่เกี่ยวข้อง
